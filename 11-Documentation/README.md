## CCNA Certification Path

### CCNA (Cisco Certified Network Associate)
- **Duration:** 3 hours
- **Questions:** 55-65 questions
- **Passing Score:** 825-875 out of 1000
- **Cost:** $330 USD
- **Validity:** 3 years

### Exam Topics

#### 1. Network Fundamentals (15%)
- OSI and TCP/IP models
- IP addressing and subnetting
- Routing and switching basics
- Network devices (routers, switches, firewalls)

#### 2. Network Access (20%)
- VLANs and trunking
- Spanning Tree Protocol
- Port security
- DHCP and ARP

#### 3. IP Connectivity (25%)
- Static routing
- Dynamic routing (OSPF, EIGRP)
- Default routes
- Floating static routes

#### 4. IP Services (10%)
- DHCP configuration
- NAT and PAT
- DNS
- SNMP

#### 5. Security Fundamentals (20%)
- ACLs
- Device hardening
- Authentication and authorization
- Network threats

#### 6. Automation and Programmability (10%)
- Python basics
- APIs
- Configuration management
- Basic automation concepts

## Networking Glossary

### Frequently Used Terms

**ACL (Access Control List)**
- Set of rules for filtering traffic
- Based on source/dest IP, protocol, port
- Applied to interfaces

**CIDR (Classless Inter-Domain Routing)**
- IP addressing scheme
- Uses prefix notation (/24, /25)
- More efficient than classful addressing

**DMZ (Demilitarized Zone)**
- Network segment separating untrusted from internal
- Contains public-facing servers
- Monitored security perimeter

**EIGRP (Enhanced Interior Gateway Routing Protocol)**
- Cisco proprietary routing protocol
- Fast convergence
- Low bandwidth usage
- Suitable for enterprise networks

**OSPF (Open Shortest Path First)**
- Open standard routing protocol
- Link-state protocol
- Scalable to large networks
- Supports ECMP load balancing

**PAT (Port Address Translation)**
- Maps multiple private IPs to one public IP
- Uses different ports for each connection
- Also called IP overloading

**STP (Spanning Tree Protocol)**
- Prevents switching loops
- Provides redundant paths
- Creates tree topology
- Root bridge election

**VLAN (Virtual Local Area Network)**
- Logical network segment
- Multiple broadcast domains on single switch
- Isolated for security
- Configured with 802.1Q tagging

**VPN (Virtual Private Network)**
- Encrypted tunnel over public network
- Site-to-site or remote access
- IPSec or SSL/TLS
- Enables secure communication
