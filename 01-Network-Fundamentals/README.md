# Network Fundamentals - Module 1

This module covers the foundational concepts of networking that every beginner must understand.

## Contents

1. **01-OSI-Model/** - Understanding the 7 layers of the OSI model
2. **02-TCP-IP-Stack/** - TCP/IP protocol suite fundamentals
3. **03-IP-Addressing/** - IPv4 and IPv6 addressing schemes
4. **04-Subnetting-Guide/** - Subnetting and CIDR notation

## Learning Objectives

By the end of this module, you will:
- ✅ Understand and apply the OSI model layers
- ✅ Explain how the TCP/IP stack works
- ✅ Design IP addressing schemes
- ✅ Perform subnetting calculations
- ✅ Understand CIDR notation and variable-length subnet masks

## Duration
**Weeks 1-2** of the project timeline

## Getting Started

Start with the OSI Model folder and work through each section sequentially. Each section builds on the previous one.

## Key Takeaways

- OSI Model provides a framework for understanding network communication
- TCP/IP is the practical implementation of networking
- IP addressing is the foundation of modern networks
- Subnetting is essential for network design and management
